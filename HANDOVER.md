# HANDOVER.md

## 1. What Was Built
A fully playable 2D platformer prototype where the player can run, jump, and stomp on enemies to reach a goal. The game includes basic physics, collision detection, enemy patrol behavior, and distinct game states for winning and dying. All core gameplay mechanics are functional and the game runs at a stable 60 FPS.

## 2. Getting Started
**Prerequisites:** Python 3.8+ and Pygame.

**Install:**
```bash
pip install pygame
```

**Run:**
```bash
python main.py
```

**Controls:** Arrow keys to move, Space to jump.
